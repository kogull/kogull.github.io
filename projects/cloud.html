Cloud

Procedure

1. Preparing the Hardware
- Install the new hard drive by opening the computer case, mounting the drive, connecting the SATA and power cables, and closing the case.

2. Installing Proxmox
- Download the Proxmox ISO file from the official website.
- Create a bootable USB drive using software like Rufus.
- Boot the computer from the USB drive and follow the on-screen instructions to install Proxmox.
- Configure the network settings and create a Proxmox cluster if needed.

3. Setting Up Virtual Machines
- Create a TrueNAS VM in the Proxmox web interface, allocate resources, attach the TrueNAS ISO, and install TrueNAS.
- Create a Docker VM, allocate resources, and install a Linux distribution (e.g., Ubuntu) on the Docker VM.

4. Installing Docker and Portainer
- SSH into the Docker VM and run the following commands to install Docker:
sudo apt update
sudo apt install docker-ce docker-ce-cli containerd.io
- Install Portainer by running the following command:
sudo docker run -d -p 9000:9000 --name=portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer-ce
- Access Portainer via http://:9000 and complete the setup.

5. Installing Jellyfin
- In Portainer, navigate to the "Containers" section, click "Add Container", and configure the Jellyfin container with the following settings:
- Image: jellyfin/jellyfin
- Ports: Map the container port 8096 to a host port.
- Volumes: Map the media library path to a host directory.
- Click "Deploy the container".
